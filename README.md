# CP 1895 Final Project - PS2 Game Tracker

##  How to Run Locally

To run the PS2 Game Vault web app on your local machine, follow these steps:

**1. Open the Project**
Open the `ps2-game-vault` folder in PyCharm.

**2. Open the Terminal**
Open PyCharm's built-in terminal (usually located at the bottom of the screen).

**3. Activate the Virtual Environment**
PyCharm typically activates this automatically (you will see `(venv)` at the start of your terminal prompt). If it is not activated, run the following command:
`venv\Scripts\activate`

**4. Install Dependencies**
Ensure the required Python libraries are installed by running:
`pip install Flask Flask-Session`

**5. Run the Application**
Start the server by running the main Python script:
`python main.py`

**6. Access the Web App**
The terminal will display the local IP address the application is running on. CTRL + Click the link below in your terminal to open the application in your default web browser:
`http://127.0.0.1:5000`
